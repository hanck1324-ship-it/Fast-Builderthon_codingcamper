# Task 2.2: 로딩/전환 오버레이 (Scene 2)

## 목표
토론 시작 시 나타나는 로딩 오버레이 구현

---

## 프롬프트

```
토론 시작 시 나타나는 로딩 오버레이를 구현해줘.

디자인 요구사항:
1. 바텀 시트 형태 (화면 80% 차지)
2. Glassmorphism 스타일:
   - background: rgba(13, 31, 60, 0.85)
   - backdrop-filter: blur(25px)
   - border: 1px solid rgba(255,255,255,0.1)
   - border-radius: 24px (상단만)
3. 드래그 핸들 (상단 중앙 회색 바)
4. 컨텐츠:
   - 제목: "AI 세미나: {강의명}"
   - 서브타이틀: "현재 수강 중: {코스명}"
   - 물결 애니메이션 SVG (중앙)
   - 제임스/린다 아바타가 물결에서 등장
   - 로딩 텍스트: "토론 패널을 초대하는 중..."
   - 3단계 프로그레스 dot

애니메이션 (Framer Motion):
1. 바텀 시트: y: 100% → 0% (spring)
2. 물결 SVG: path morphing 애니메이션
3. 아바타: scale 0 → 1, stagger 0.2초
4. 로딩 텍스트: 3단계 페이드 전환
   - "강의 컨텍스트 분석 중..."
   - "토론 패널을 초대하는 중..."
   - "AI 세미나 준비 완료!"

Props:
- lectureTitle: string
- courseName: string  
- onLoadingComplete: () => void
```

---

## 애니메이션 타임라인

```
0.0s ─── 바텀 시트 슬라이드 업
0.3s ─── 물결 애니메이션 시작
0.5s ─── "강의 컨텍스트 분석 중..."
1.2s ─── 제임스 아바타 등장
1.4s ─── 린다 아바타 등장
1.5s ─── "토론 패널을 초대하는 중..."
2.5s ─── "AI 세미나 준비 완료!"
3.0s ─── onLoadingComplete() 호출
```
